## 🌐 Descripción

**Yoga Studio Pro** es una plataforma web desarrollada por **AG Soluciones de Ingeniería en Software**, diseñada para academias, estudios de yoga, gimnasios y centros de bienestar que desean digitalizar la administración de sus clases, alumnos y reservas.

El sistema automatiza procesos administrativos, mejora la experiencia de los clientes y permite gestionar el negocio desde cualquier dispositivo.
